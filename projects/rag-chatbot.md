# 🔍 RAG Chatbot Project (Retrieval Augmented Generation)

## 📌 Overview
This project explains how a RAG (Retrieval Augmented Generation) system works. It enhances Large Language Models by allowing them to retrieve information from external documents before generating answers.

---

## 🧠 How it works
1. User asks a question
2. System searches relevant documents (vector search)
3. Retrieved context is passed to LLM
4. LLM generates accurate response using context

---

## ⚙️ Key Concepts
- Embeddings
- Vector databases (FAISS concept)
- Semantic search
- Prompt augmentation
- Context injection into LLM

---

## 🧩 Architecture (High Level)
User Query → Embedding Model → Vector DB Search → Retrieved Docs → LLM → Final Answer

---

## 🚀 Why RAG is important
- Reduces hallucinations
- Improves factual accuracy
- Allows LLMs to use private data
- Used in real-world AI chatbots
